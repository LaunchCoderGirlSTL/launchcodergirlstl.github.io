CoderGirl Home

learn.launchcode.org is the home of content for LaunchCode's CoderGirl program. You must be enrolled in an active section of CoderGirl to access the site.

Project Cycle

The Project Cycle is for learners that have completed a CoderGirl learning track.

This site collects topic-specific resources for building projects. It also contains resources for career modules: Resumes and Online Profiles, Technical Interviews, and Live Coding.

User Experience

User Experience is the process of designing (digital or physical) products that are easy to use and delightful to interact with. It’s about enhancing the experience that people have while interacting with your product, and making sure they find value in what you’re providing.

Front-End Web Dev

Front-End Web Development covers the basics of web development using HTML, CSS, and JavaScript, how to build responsive websites for mobile through desktop, and get prepared to learn additional web frameworks like Angular, Ember, and React.

Java

Java is an object-oriented language with a multitude of applications. In this course you will start with the fundamentals of programming and progress through the Java language as you learn to solve problems with code. Java developers work with data management, reporting, and business workflows.

C#

C# is used primarily on Windows-based machines and has been widely adopted by companies around the world as their software standard. The .NET Framework is a combination of code and environment that provides a common runtime platform.

iOS

iOS focuses primarily on the Foundation Framework and UIKit, the fundamental, yet massive, framework provided by Apple to design, create, animate, and customize your entire apps experience. This course is great for those with some object oriented programming experience and ready to jump into the mobile field.

SQL

SQL is the most widely used tool for working with databases. This class begins with the basics of SQL and moves toward advanced queries and reporting. It is for anyone interested in working with databases, including programmers and non-programmers alike.

Data Science

Data Science can be applied to discovering customer insights, informing digital marketing, scientific fields of research and application of predicting outcomes or optimizing processes. This course will give you a comprehensive overview of data science methods and tools.

Data Analysis

Data Analysis is taught in coordination with the Data Science track. Both tracks share the first 6 weeks of curriculum, at which point students may continue learning one or the other. Data Analysis is more focused on working with, drawining insights from, and communicating knowledge from data sets.

GIS

The CoderGirl GIS curriculum introduces learners with a basic programming skill set to GIS development concepts.

After an introduction to JavaScript, it introduces the OpenLayers GIS JavaScript framework. It then explores using GeoServer to create and customize GIS data stores.
